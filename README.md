# Web-Project-Danawa
다나와 사이트를 참고하여 게시판과 노트북 상세 검색을 구현한 웹 프로젝트입니다.
</br></br>
  
## 개발 환경 및 사용 기술
OS: Window  
IDE: Eclipse  
Front: JavaScript, jQuery, Thymeleaf  
Back: Java 11, SpringBoot, Gradle, Spring Data JPA, H2 Database  
크롤링: Selenium
</br></br>
    
## 주요 기능
노트북 상세 검색 (옵션 체크, 키워드 검색, 가격 범위 설정)  
노트북 가격 비교  
노트북 관심 상품 등록  
회원가입, 로그인  
게시판 글 작성, 수정, 삭제, 이미지 첨부  
게시물 추천 기능, 댓글 기능  
페이징, 정렬, 검색 기능
</br></br>
